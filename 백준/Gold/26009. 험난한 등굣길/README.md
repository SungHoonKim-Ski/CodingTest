### 출력 

 <p>재헌이가 지각하지 않고 수업을 들을 수 있으면 <code style="color:#e74c3c;">YES</code>를 출력하고, 다음 줄에 최소 이동 횟수를 출력한다.</p>

<p>만약 지각하지 않고 수업을 들을 수 없다면 <code style="color:#e74c3c;">NO</code>를 출력한다.</p>
